# 🎯 Guess Number Game (Python)

A simple Python console game where the player has to guess a randomly generated number between **1 and 200**.

## Features

- Random number generation
- 6 guessing attempts
- Input validation
- User-friendly messages
- Play Again option

## Requirements

- Python 3.x

## Run

```bash
python guess_number.py
```
